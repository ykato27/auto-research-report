# タレントマネジメント・インテリジェンス 週次ニュースまとめ
─ Claude Code Web 定期実行プロンプト

## 概要

| 項目 | 内容 |
|---|---|
| テーマ | タレントマネジメント・インテリジェンス（国内・海外） |
| 頻度 | 毎週月曜日 |
| 推奨実行時間 | 日本時間 月曜 朝 8:00 |
| 送信先 | yuki.kato@skillnote.co.jp |
| 送信方法 | send_daily_ai_news.py（SMTP） |
| 対象範囲 | 直近7日間（国内・海外英語ソース） |
| 目標件数 | 40〜60件 |

## 登録手順

1. claude.ai/code にアクセス
2. リポジトリ `ykato27/auto-research-report` を選択
3. スケジュール設定画面で下記プロンプトを登録
4. 頻度を「毎週」、時間を日本時間 月曜 朝 8:00 に設定
